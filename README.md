## 3. Описание предметной области
RSA (Rivest–Shamir–Adleman) — криптосистема с открытым ключом, основанная на вычислительной сложности задачи факторизации произведения двух больших простых чисел. 
В данной работе реализована упрощенная версия алгоритма:
1. Выбираются два простых числа $p$ и $q$.
2. Вычисляется модуль $n = p \times q$ и функция Эйлера $\phi(n) = (p-1)(q-1)$.
3. Выбирается открытая экспонента $e$.
4. Вычисляется секретная экспонента $d$ (обратное число к $e$ по модулю $\phi(n)$).
5. Текст шифруется посимвольно по формуле $c = m^e \pmod{n}$.

1.  **Модульное тестирование (Unit Testing):** Проверка отдельных методов шифрования без запуска UI.
2.  **Интеграционное тестирование:** Проверка взаимодействия класса `RSACipher` с интерфейсом WPF.

4.  **Техника «Белого ящика»:** Отладка кода внутри метода `ModInverse` с помощью точек останова (Breakpoints).

**Итог:** Приложение стабильно работает со всеми типами символов (ASCII, Unicode), корректно обрабатывает исключительные ситуации и предоставляет пользователю обратную связь через всплывающие подсказки и сообщения об ошибках.
